# playerstatsleaderboardlua
Script for loading in player stats in Lua Roblox. It's great for displaying player game/experiences through leaderstats/playerstats.
May not work as expected on Roblox Studio. For the best experience, I suggest using it in Roblox to playtest.

REQUIRED: Allow HTTP Requests on your game.
SUGGESTED: Use CustomLeaderboardUI script to elevate the experience and to bypass the integer limit.
